Objective

To analyze the factors that influence student academic performance using Exploratory Data Analysis (EDA), and to build a Linear Regression model that predicts a student's final score based on behavioral, academic, and lifestyle attributes from a Kaggle dataset.

Project Steps — Complete Sequence
1

Dataset Loading

Loaded the student performance dataset from Kaggle using Python Pandas. The dataset contains academic, behavioral, and lifestyle attributes across 1,000+ student records with multiple feature columns representing different aspects of student life and learning habits.

PandasCSV LoadKaggle Dataset
2

Initial Data Exploration

Explored dataset structure, data types, null values, and statistical summaries to understand the data before analysis. Key functions used:

  • head() — Preview the first few rows of the dataset
  • shape() — Check total number of rows and columns
  • info() — Review data types and identify missing values
  • describe() — Generate statistical summary (mean, std, min, max)
Data ProfilingEDA
3

Feature Selection

Identified and selected 6 key attributes most likely to impact student final scores, used as input features for the prediction model:

  • previous_score — Prior academic performance
  • attendance_percentage — Class attendance rate
  • daily_study_hours — Average daily study time
  • math_prev_score — Previous Mathematics score
  • science_prev_score — Previous Science score
  • language_prev_score — Previous Language score
Feature EngineeringScikit-Learn
4

Exploratory Data Analysis (EDA)

Computed a correlation matrix across all numerical columns to identify relationships between features. Generated a Seaborn heatmap to visually observe which variables have the strongest positive or negative correlation with the final score — helping prioritize the most influential features for the model.

Correlation MatrixSeaborn HeatmapNumPy
5

Data Visualization

Created multiple visualizations to analyze how different factors influence student performance:

  • Regression Plot — Previous score vs. Final score relationship
  • Regression Plot — Attendance percentage vs. Final score relationship
  • Boxplot — Impact of study environment on overall academic performance
MatplotlibSeabornRegression PlotsBoxplots
6

Machine Learning — Linear Regression Model

Built a Linear Regression model using Scikit-Learn to predict each student's final_score based on the 6 selected features. Split the dataset into training and testing sets, trained the model on historical data, and evaluated prediction performance using standard regression metrics.

Linear RegressionScikit-LearnTrain/Test SplitPredictive Modeling
7

Insights & Conclusions

Identified the following key patterns from the analysis to support data-driven recommendations for improving student academic outcomes:

  • Previous academic scores and attendance percentage showed the strongest positive impact on final performance
  • Daily study hours had a moderate positive correlation with final score
  • Study environment had a significant influence on overall academic performance
  • Students with consistent study habits consistently outperformed peers with irregular patterns

Key Findings
📚

Previous Scores Matter Most

Prior academic performance was the strongest predictor of final scores, showing that consistent past performance strongly predicts future outcomes.

🏫

Attendance is Critical

Higher attendance percentage was directly linked to better final scores across all subjects and student demographics in the dataset.

⏰

Study Hours Impact Performance

Daily study hours showed a moderate positive correlation with final scores, reinforcing the value of consistent and structured study habits.

🏠

Study Environment Matters

Students in a favorable and distraction-free study environment consistently outperformed those in less conducive learning settings.
